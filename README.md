## Tech Stack & Architecture

- **Primary Technologies**: Python (Standard Library only - strictly array mapping parsing).
- **Core Concepts**: Heuristic search matrices, node graphing algorithms, and BFS state queue tracking.

**High-Level Architecture**:
The foundational system utilizes sequential 2D arrays parsing boundary logic (`0`, `1`, `2` layouts) to simulate navigable grids vs impassable edges logically resolving paths.
- **Node Matrix (`TreeNode`)**: Scalable object hierarchy linking generated exploration layers directly.
- **State Generation Engine (`generate_next_state`)**: Mutates grid matrices parsing Cartesian dimensional arrays (`[[-1,0], [1,0], [0,-1], [0,1]]`) to dynamically discover viable steps cleanly.
- **Traversal Flow**: Instantiates linear memory tracking referencing `visited_states` iteratively.
