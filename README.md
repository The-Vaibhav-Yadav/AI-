# AI Laboratory Experiments

This repository contains base experiments and assignments pertaining to Artificial Intelligence coursework.

## Contents

- **`lab1.py`**: Explores a breadth-first search (BFS) algorithm to solve a pathfinding or puzzle scenario on a matrix map. It defines a `TreeNode` representation for state nodes and implements generic moves/state generation for exploration.

### Running

```bash
python3 lab1.py
```
